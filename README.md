MSWL_Development_Tools_GTK
==========================

GTK examples

gPad - Text Editor
    * A copy of Mousepad
            
gImageViewer - Image Viewer
    * A copy of GNOME image viewer


Useful commands:
* A developers' help browser for GNOME
$ devhelp
* GTK+ Code Demos - Program to demonstrate GTK+ functions. 
$ gtk3-demo
